Settings

TSettings is basically a set of variables that the underlying layers are using. All the variables have got default set values. It's marked !default so that the preprocessor build, will say this: "If no other value is set to this variable by now, this is what it'll get."

Base variables

Breakpoints

$phone:             0 !default;
$tablet:            37.5rem !default;
$tablet-landscape:  56.25rem !default;
$desktop:           64rem !default;
$desktop-lg:        85.375rem !default;
$desktop-xl:        120rem !default;
